## Editing Resume Data

### Basic Workflow

1. **Open your portfolio** (visit the shareable link)
2. **Click the "Edit" button** (top left, gray button)
3. **A form modal opens** with all your portfolio content
4. **Make changes** to any fields
5. **Click "Save Changes"** (bottom right)
6. ✅ **Changes saved instantly!**

### What You Can Edit

#### ✏️ **Basic Info**
- Full name
- Email address
- Phone number
- Location
- LinkedIn URL
- GitHub URL
- Professional summary

#### ✏️ **Work Experience**
- Job title
- Company name
- Start date
- End date (or "Present" if current)
- Description (bullet points - one per line)
- Add/remove experience entries

#### ✏️ **Education**
- Degree type
- University/Institution
- Field of study
- Graduation year
- Add/remove education entries

#### ✏️ **Skills**
- Add/remove skills (comma-separated)
- Example: `Python, React, Node.js, TypeScript, AWS`

#### ✏️ **Projects**
- Project name
- Description
- Technologies used
- Project URL
- Add/remove projects

#### ✏️ **Certifications**
- Add/remove certificates

## Feature 1: Profile Photo Upload

### What It Does
Adds a professional headshot to your portfolio that displays prominently in all three templates.

### How to Upload a Photo

1. **Click "Edit"** to open the editing modal
2. **Scroll to the top** - you'll see "Profile Photo" section
3. **Click "Upload Photo"** button
4. **Select an image** (JPEG, PNG, or WebP)
5. **Wait for upload** (max 5 MB)
6. ✅ **Photo appears in preview** and on your live portfolio

### Photo Display by Template

- **Minimal Template**: Centered circle avatar above your name
- **Modern Template**: Circle profile pic in the dark sidebar above your name
- **Creative Template**: Circle photo in the colorful hero gradient section

### Best Practices

- Use a **high-quality headshot** (professional photo)
- Square images work best (will be cropped to circle)
- Size: 500x500px or larger
- File size: Under 5 MB (usually ~200-500 KB is fine)
- Formats: JPEG, PNG, or WebP

### Re-uploading a Photo

Just click "Upload Photo" again with a new image. It will replace the old one.

## Feature 2: Custom Colors

### What It Does
Customize the primary accent color and background color of your portfolio to match your brand.

### How to Change Colors

1. **Click the "Settings" button** (gear icon, top right)
2. **Settings Panel opens** with color pickers
3. **Primary Accent Color**: Click the color square to pick (or paste hex code)
   - Used for section headings, links, accents
   - Default: `#3b82f6` (blue)
4. **Background Color**: Pick the background tone
   - Default: `#ffffff` (white)
5. **Custom URL Slug** (optional - see Feature 7)
6. **See preview** at the bottom of the panel
7. **Click "Save Settings"** ✅

### Default Colors

```
Primary Accent: #3b82f6 (Blue)
Background: #ffffff (White)
```

### Color Application by Template

- **Minimal Template**: Accent color on all section headings
- **Modern Template**: Sidebar becomes accent color; headings and company names use accent
- **Creative Template**: Section labels and accent elements use custom color

### Popular Color Combinations

```
Professional Blue: #3b82f6 (accent) + #ffffff (background)
Tech Purple: #7c3aed (accent) + #f3f4f6 (background)
Corporate Teal: #0891b2 (accent) + #f8fafc (background)
Creative Orange: #ea580c (accent) + #fef3c7 (background)
```

### Hex Code Resources

- [Colorhexa](https://www.colorhexa.com) - Color picker & info
- [Coolors.co](https://coolors.co) - Generate color palettes
- [Material Design Colors](https://material.io/resources/color) - Pre-made palettes

## Feature 3: Drag-to-Reorder Sections

### What It Does
Customize the order that sections appear on your portfolio (Experience, Education, Projects, etc.).

### How to Reorder Sections

1. **Click "Edit"** to open the editing modal
2. **Look for "Section Order"** (blue box at the top)
3. **See all 5 sections** listed:
   - Experience
   - Education
   - Projects
   - Skills
   - Certifications
4. **Use up/down arrows** to move sections
5. **Changed order is highlighted** (blue button appears: "Save Section Order")
6. **Click "Save Section Order"** ✅

### Example: Reorder for Tech Roles

Want to highlight projects first? Reorder like this:

```
1. Projects      (employers care about what you've built)
2. Experience    (then your work history)
3. Education
4. Skills
5. Certifications
```

### Example: Reorder for Academics

```
1. Education     (degree is most important)
2. Projects      (research/thesis work)
3. Experience    (internships/assistantships)
4. Skills
5. Certifications
```

### Note

- This saves independently from the main "Save Changes" button
- Each template respects your custom order
- If not set, defaults to: Experience → Education → Projects → Skills → Certifications

## Feature 4: Dark/Light Mode

### What It Does
Toggle between light and dark theme for your portfolio. Dark mode reduces eye strain and looks modern.

### How to Enable Dark Mode

1. **Click the sun/moon icon** in the top right (next to Settings)
2. **Button changes color**:
   - Yellow background = Light mode
   - Dark gray background = Dark mode
3. ✅ **Portfolio instantly switches**

### Template Support

| Template | Light Mode | Dark Mode |
|----------|-----------|-----------|
| **Minimal** | ✅ Full support | ✅ Full support |
| **Modern** | ✅ Full support | ✅ Full support |
| **Creative** | ⚫ Always dark | ⚫ Always dark |

### Light Mode (Default)

- White background
- Dark text
- Clean, professional appearance
- Better for printing

### Dark Mode

- Dark gray/black background
- Light text
- Modern, eye-friendly
- Popular for tech portfolios
- Better for night viewing

### Persistence

- **Dark mode setting is saved** to your portfolio
- **Survives page refreshes**
- **Applied automatically** when sharing the link
- Recruiters see your chosen theme

## Feature 5: Preview Mode

### What It Does
Hide all editing controls and see your portfolio as recruiters see it (recruiter view). Great for:
- Double-checking appearance before sharing
- Testing how it looks without edit buttons
- Demonstrating to others

### How to Use Preview Mode

1. **Click "Preview" button** (eye icon, top right)
2. **All controls disappear**:
   - Edit button hidden
   - Settings button hidden
   - Template switcher hidden
   - ATS sidebar hidden
3. **Portfolio fills entire screen** (clean recruiter view)
4. **Floating "Exit Preview" button** appears (bottom right)
5. **Click "Exit Preview"** to go back to editing

### Use Cases

- **Before sharing**: Make sure everything looks good
- **Demo mode**: Show stakeholders the final result
- **Mobile preview**: Test mobile appearance without distractions
- **Screenshot**: Take clean screenshots for marketing

## Feature 7: Custom Portfolio Slug

### What It Does
Create a memorable custom URL for your portfolio (e.g., `/p/johndoe` instead of `/p/abc123def456`).

### How to Set a Custom Slug

1. **Click "Settings"** (gear icon, top right)
2. **Scroll to "Custom URL Slug"** section
3. **Enter your desired slug** (see rules below)
4. **Green checkmark ✓** = Slug is available
5. **Red X ✗** = Slug is taken or invalid
6. **Click "Save Settings"** ✅

### Slug Rules

- **Length**: 3-50 characters
- **Characters**: Lowercase letters, numbers, hyphens only (no spaces, underscores, uppercase)
- **Format**: Must start and end with letter/number (no hyphens at edges)

### Valid Examples

```
✅ johndoe
✅ jane-smith-dev
✅ sarahdev2024
✅ tech-recruiter-nyc
```

### Invalid Examples

```
❌ john doe        (has space)
❌ John-Doe        (has uppercase)
❌ -john           (starts with hyphen)
❌ jo               (too short, needs 3+ chars)
```

### Your Custom URL

Once saved, your portfolio is accessible at:

```
https://youromain.com/p/johndoe
```

### Benefits

- **Easy to remember** for recruiters
- **Shareable in emails** without long URL
- **Professional appearance**
- **Works globally** like regular links

### Changing Your Slug

Just open Settings and change it anytime. The old slug becomes unavailable for others to use.

## Feature 8: QR Code Generator

### What It Does

Generate a scannable QR code that links to your portfolio. Perfect for sharing on business cards, printed resumes, and at networking events.

### How to Get Your QR Code

1. **Click "QR Code"** (grid icon, top right)
2. **A modal appears** with your QR code
3. **See your portfolio URL** displayed below the code
4. **Click "Download"** to save as PNG image
5. **Click "Close"** to dismiss modal

### Downloading Your QR Code

The QR code downloads as a PNG image with:
- **Filename**: `portfolio-{slug}.png` or `portfolio-{id}.png`
- **Size**: 232x232 pixels (includes padding)
- **Format**: High quality PNG for printing

### Using Your QR Code

**On Business Cards:**
- Print in corner (about 1" × 1")
- Add label: "Scan to view portfolio"

**On Printed Resume:**
- Add to header or footer
- Include small text: "Scan for interactive portfolio"

**At Conferences/Networking Events:**
- Display on phone when handing out cards
- Have a poster-sized version printed

### QR Code Details

- Works with **both slug and UUID URLs**
- If you have a custom slug: QR points to `/p/your-slug`
- If no slug: QR points to `/portfolio/{your-id}`
- QR scans work on **all smartphones** (no app needed)
- Recruiters see your **latest portfolio version** when they scan

### Tips

- Make sure your slug is memorable and professional before printing
- Test the QR code with your phone to verify it works
- Download multiple copies if you need different sizes
- Light-colored resumes work best (high contrast with QR code)

## Social Sharing: Open Graph Preview

### What It Does

When you share your portfolio link on **LinkedIn, Twitter, WhatsApp, Facebook**, or other social platforms, instead of showing a blank card, it displays a beautiful preview with:
- Your name and headline
- Your professional summary/bio
- Your profile photo
- A link to your portfolio

This **dramatically increases click-through rates** — studies show links with preview cards get **3× more clicks** than plain URLs.

### How It Works

You don't need to do anything! The preview is **automatic**:

1. **Share your portfolio link** anywhere on social media
2. **The platform fetches your data** (name, summary, photo) from your portfolio
3. **A rich preview card appears** showing your info
4. **Recruiters and connections click** the preview to view your full portfolio

### What Shows in the Preview

The preview includes:
- **Title**: Your name + "— Portfolio"
- **Description**: Your professional summary (from resume)
- **Image**: Your profile photo (if you uploaded one)
- **Link**: Your portfolio URL (slug or UUID)

### Testing the Preview

To see how your portfolio preview looks before sharing:

1. **LinkedIn**: Use the [LinkedIn Post Inspector](https://www.linkedin.com/feed/) — paste your link in a comment or post
2. **Twitter/X**: Use the [Twitter Card Validator](https://cards-dev.twitter.com/validator)
3. **General**: Use [opengraph.xyz](https://www.opengraph.xyz) — paste your portfolio URL

### Tips for Best Preview

- **Add a profile photo** — Preview looks much more professional with your photo
- **Write a good summary** — This text appears in the preview (from your original resume)
- **Use a custom slug** — `/p/johndoe` looks better in preview than `/portfolio/abc123`
- **Share after editing** — Portfolio preview updates in real-time as you edit your info

### Example

When you share `https://myportfolio.com/p/johndoe`:

**LinkedIn Preview Shows:**
```
John Doe — Portfolio
Experienced software engineer with 5+ years building scalable web apps.
[John's photo]
myportfolio.com/p/johndoe
```

Then people click the preview and see your full interactive portfolio!

## View Count Analytics

### What It Shows

In the header bar on your portfolio page (left side, next to "← New portfolio"), you'll see a view counter like:

```
👁 12 views
```

This shows **how many times your portfolio has been accessed** since you created it.

### How It Works

Every time someone:
- Clicks your share link (`/portfolio/{id}` or `/p/{slug}`)
- Views your portfolio in their browser

The view count increases by 1. The counter is displayed **only to you** — recruiters who view your portfolio don't see it.

### Why This Matters

**Motivation to Share:** The view count is concrete feedback that recruiters are clicking your link. This motivates you to:
- Share more actively on LinkedIn, email, etc.
- Refine your portfolio based on interest
- Track which sharing channels work best

**Job Application Tracking:** When you apply to a job, you can:
1. Note the view count before sending your link
2. Share the link with recruiters
3. Check back to see if they viewed it

### Tips

- **Cold outreach:** Send your portfolio link to recruiters and check if they view it
- **Track campaigns:** Screenshot the view count when sharing to track response
- **Refine content:** High views + low interest? Update your summary or skills
- **Share timing:** Notice patterns in when people view your portfolio

### Note

The view count is **reset if you create a new portfolio**. Each portfolio is independent with its own view counter starting at 0.

## FAQs & Tips

### Q: Will the link break if I edit?
**A:** No! Your link stays the same. Recruiters see your latest changes instantly.

### Q: Can I have multiple portfolios?
**A:** Yes! Upload a resume again to create a new portfolio with a different slug (e.g., `johndoe-cv2`).

### Q: Can I undo changes?
**A:** Not yet, but you can manually revert in the edit form. Save a backup slug combo if making major changes.

### Q: Does editing recalculate the ATS score?
**A:** No. ATS score is locked to your original resume. Upload a new resume to recalculate.

### Q: Can I share my slug URL with anyone?
**A:** Yes! Once you set a slug, it creates a public-facing URL that anyone can visit globally.

### Q: What if my slug is taken?
**A:** The Settings panel will show a red X. Try a variation (e.g., `johndoe2024` instead of `johndoe`).

### Q: Can I change my template after editing?
**A:** Yes! Switch templates anytime—your edits apply to all of them.

### Q: Do my colors/dark mode apply to the public link?
**A:** Yes! All customizations (colors, dark mode, section order, badge, photo) are saved to your portfolio and visible to anyone with the link.

### Q: Can I have different colors per template?
**A:** No, colors apply globally. But each template looks different with the same color.

### Q: Is there a character limit for descriptions?
**A:** Not enforced, but keep experience bullets under 200 characters each for readability.
